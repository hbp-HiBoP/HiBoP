Detailed Description

Data of the visualisation Column.

Author
Adrien Gannerie
Version
1.0
Date
10 janvier 2017

ColumnData is a class which contains all the data of the visualisation Column and contains:

  • Label.
  • Mask of the plots.
  • Time line.
  • Iconic scenario.
  • Values.
