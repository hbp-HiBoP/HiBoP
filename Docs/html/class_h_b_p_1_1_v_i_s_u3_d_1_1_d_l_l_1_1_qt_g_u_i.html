Detailed Description

A singleton interface class which crate a QApplication and allow to communicate with Qt gui functions like QFileDialog.

Examples :

string dirPath = qtGui.getExistingDirectory("Select a directory "); string filePath = qtGui.getOpenFileName(new string[] { "txt", "exe", "png" }, "Select a file", ""); string[] filesPaths = qtGui.getOpenFilesName(new string[] { "txt", "exe", "png" }, "Select files", ""); string saveFilePath = qtGui.getSaveFileName(new string[] { "txt", "exe", "png" }, "Save file to", "");
