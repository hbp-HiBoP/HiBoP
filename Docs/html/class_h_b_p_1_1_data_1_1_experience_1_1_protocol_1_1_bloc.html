Detailed Description

Bloc in a Protocol.

Author
Adrien Gannerie
Version
1.0
Date
05 janvier 2017

Class which define a bloc in a visualisation protocol which contains :

  • Display informations.
  • Main event.
  • Secondary events.
  • Iconic scenario.
  • Unique ID.
