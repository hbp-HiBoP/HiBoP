Detailed Description

Brain anatomic data.

Author
Adrien Gannerie
Version
1.0
Date
03 janvier 2017

Brain anatomic data which contains:

  • Meshes.
  • IRMs.
  • Electrodes implantations.
  • Base transformations.
  • Epilepsy.
